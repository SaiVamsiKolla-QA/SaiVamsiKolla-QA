## About Me

I'm a QA engineer with 7+ years of experience. I started in manual testing, moved into automation and API testing, and now spend most of my time validating AI systems.

Right now I'm the only QA engineer on an on-device AI health app at mimik. I set up the test strategy, built the first automation framework for the product, and test how the on-device AI agent actually behaves.

Before mimik I tested autonomous AI checkout systems at Meta (through HCLTech). Most of my earlier career was in Guidewire and P&C insurance, leading integration, UAT, and automation work on big implementations.

## Featured Projects

**[microservice-api-testing](https://github.com/SaiVamsiKolla-QA/microservice-api-testing)**
A Python API testing framework built against the Swagger Petstore (OpenAPI v3). Uses httpx, pydantic v2, pytest, schemathesis for fuzz testing, and allure-pytest for reporting. Managed with Poetry.

**[qa-expert-agent](https://github.com/SaiVamsiKolla-QA/qa-expert-agent)**
A local RAG-based QA mentor agent that runs fully on-device using Python, ChromaDB, and a small on-device model (SmolLM2-360M). Phase 1 shipped with 59 passing tests and documented evaluation results.
